# 乡村振兴法律咨询 Demo

一个本地可运行的小型演示项目：基于法条片段做检索（RAG），再调用大模型生成综合答复。

## 已实现功能
- 法条检索：对 `data/laws/chunks.jsonl` 做关键词检索（BM25），并结合主题词做召回增强。
- RAG 生成：把命中的法条片段拼接为上下文，调用大模型生成整体答复与建议。
- SSE 流式输出：前端可以边生成边显示。
- 基本前端展示：问题/回答卡片、法条卡片、示例一键填充。

## 运行方式
1. 准备环境变量（可选）
   - 复制 `.env.example` 为 `.env`
   - 填写 `LLM_API_KEY`、`LLM_MODEL`、`LLM_BASE_URL` 等
2. 启动服务
   ```bash
   python demo_server.py
   ```
3. 浏览器访问
   - http://localhost:8000

## 目录结构
- `demo_server.py`：后端服务（检索 + 调用大模型）
- `demo/`：前端页面与脚本
- `data/laws/chunks.jsonl`：法条切分后的检索数据
- `scripts/`：数据处理脚本
- `Chinese-Laws/`：法条原始文件

## 说明
- `.env` 已加入 `.gitignore`，不会提交到仓库。
- 本项目为演示用途，输出不构成法律意见。
